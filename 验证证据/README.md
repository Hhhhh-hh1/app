## 反例分析要点

反例 `6_counter_example_bokeh.jpg` 的主体区域可能是清晰的，但背景存在大面积景深虚化。当前 App 使用全图 Laplacian 方差评价清晰度，大面积低纹理背景会拉低整体边缘能量，因此可能把“有意虚化”误判为“整体模糊”。这个反例能说明算法可解释，但不是万能审美模型。
